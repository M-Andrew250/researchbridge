Overview

About This Course

IBM SPSS Statistics is one of the most widely used statistical software packages in academic research, healthcare, social sciences, and market research. Its point-and-click interface makes it accessible to researchers without programming backgrounds while still offering powerful statistical capabilities.

This course takes you from data entry and cleaning through to advanced statistical tests and reporting — with every session grounded in real research scenarios and datasets familiar to social science researchers.

Course Outline

Module Breakdown

Module 1 SPSS Foundations +
  • Introduction to the SPSS interface
  • Data View vs Variable View explained
  • Defining variables — name, type, label, values
  • Entering and importing data from Excel and CSV
  • Saving and managing SPSS files
Module 2 Data Cleaning & Transformation +
  • Identifying and handling missing values
  • Recoding variables and creating categories
  • Computing new variables from existing ones
  • Sorting, selecting, and filtering cases
  • Splitting files and merging datasets
Module 3 Descriptive Statistics & Charts +
  • Frequencies, descriptives, and explore
  • Cross-tabulations and contingency tables
  • Charts — bar, pie, histogram, box plot
  • Editing charts in the Chart Editor
  • Exporting output to Word and Excel
Module 4 Inferential Statistics +
  • Independent samples and paired t-tests
  • One-way and two-way ANOVA
  • Chi-square test of independence
  • Pearson and Spearman correlation
  • Interpreting p-values and effect sizes
Module 5 Regression & Advanced Tests +
  • Simple and multiple linear regression
  • Binary logistic regression
  • Reliability analysis — Cronbach's alpha
  • Factor analysis basics
  • Writing up results in APA format

FAQ

Frequently Asked Questions

Do I need SPSS installed before the course? +

Yes. SPSS is paid software but IBM offers a free 30-day trial. We will guide you through installation and licensing options before the first session.

Do I need prior statistics knowledge? +

Basic knowledge is helpful but not required. We explain the statistical concepts alongside the SPSS steps so you understand both the tool and the method behind it.

Will I get a certificate? +

Yes. All participants who complete the course and practical assessment receive an official ResearchBridge Certificate of Completion.

Can my organisation book this for a team? +
