Overview

About This Course

NVivo is the world's leading software for qualitative and mixed methods research. It helps researchers organise, code, and analyse unstructured data — interviews, focus groups, documents, social media, videos, and more — with rigour, transparency, and efficiency.

This course takes you from setting up your first NVivo project through to advanced thematic coding, querying, and visualisation — giving you the skills to conduct credible, defensible qualitative research for academic or professional purposes.

Course Outline

Module Breakdown

Module 1 NVivo Foundations +
  • Introduction to qualitative research and NVivo
  • Setting up and navigating an NVivo project
  • Importing documents, PDFs, and transcripts
  • Importing audio, video, and image files
  • Organising sources into folders and sets
Module 2 Coding & Node Management +
  • Introduction to coding — what and why
  • Creating and applying nodes to text
  • Building hierarchical node structures
  • In vivo coding and descriptive coding
  • Reviewing and refining your coding framework
Module 3 Qualitative Analysis Approaches +
  • Thematic analysis in NVivo
  • Framework analysis and matrix coding
  • Grounded theory coding in NVivo
  • Case nodes and attributes for comparison
  • Memos and annotations for analytical notes
Module 4 Queries & Exploration +
  • Text search and word frequency queries
  • Coding queries — retrieving coded content
  • Matrix coding queries for cross-case comparison
  • Compound queries combining multiple criteria
  • Saving and reusing queries
Module 5 Visualisation & Reporting +
  • Word clouds and word trees
  • Tree maps and cluster analysis charts
  • Concept maps and project maps
  • Exporting nodes, queries, and charts
  • Writing up qualitative findings transparently

FAQ

Frequently Asked Questions

Do I need NVivo installed before the course? +

Yes. NVivo offers a free 14-day trial which is sufficient for the course duration. We will guide you through installation and setup before the first session begins.

Do I need qualitative research experience? +

Basic familiarity with qualitative research is helpful but not required. We cover the key qualitative concepts alongside the NVivo tools throughout the course.

Will I get a certificate? +

Yes. All participants who complete the course and practical assessment receive an official ResearchBridge Certificate of Completion.
