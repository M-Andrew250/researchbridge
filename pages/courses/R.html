Overview

About This Course

R is the world's leading open-source language for statistical computing and data visualisation. Freely available and backed by a vast community of researchers and data scientists, R is used in universities, research institutions, central banks, and international organisations globally.

This course takes you from R basics through to advanced statistical modelling and publication-quality visualisation using the tidyverse and ggplot2 — with every session grounded in real research datasets and reproducible R script writing.

Course Outline

Module Breakdown

Module 1 R Foundations +
  • Installing R and RStudio
  • R syntax — objects, vectors, and functions
  • Data types — numeric, character, logical, factor
  • Data structures — vectors, lists, data frames
  • Installing and loading packages
Module 2 Data Wrangling with tidyverse +
  • Importing data with readr and readxl
  • Filtering, selecting, and arranging with dplyr
  • Creating and transforming variables with mutate
  • Grouping and summarising with group_by and summarise
  • Reshaping data with tidyr — pivot_longer and pivot_wider
Module 3 Data Visualisation with ggplot2 +
  • The grammar of graphics — layers and aesthetics
  • Bar charts, histograms, and density plots
  • Scatter plots, line graphs, and box plots
  • Faceting — small multiples for comparisons
  • Themes, colours, and exporting publication figures
Module 4 Statistical Analysis +
  • Descriptive statistics and distribution analysis
  • T-tests, ANOVA, and chi-square tests
  • Correlation analysis — Pearson and Spearman
  • Linear regression with lm() and model diagnostics
  • Logistic regression with glm()
Module 5 Reproducible Research with R Markdown +
  • Introduction to R Markdown and knitr
  • Combining text, code, and output in one document
  • Knitting to PDF, Word, and HTML
  • Creating automated research reports
  • Best practices for reproducible research workflows

Frequently Asked Questions

Is R free to use? +

Yes. R and RStudio are completely free and open-source. You can download both before the course starts and we guide you through setup in the first session.
