Overview

About This Course

Python is the world's most popular programming language for data science, research, and automation. This course takes you from zero programming knowledge to confidently writing Python scripts for data analysis, visualisation, and basic machine learning.

Every session is practical and project-based — you work with real datasets from research, business, and development contexts so that learning feels immediately relevant to your work.

Course Outline

Module Breakdown

Module 1 Python Foundations +
  • Setting up Python and Jupyter Notebook
  • Variables, data types, and operators
  • Lists, tuples, dictionaries, and sets
  • Control flow — if statements and loops
  • Writing and calling functions
Module 2 Data Handling with Pandas +
  • Loading CSV, Excel, and JSON files
  • Exploring and summarising DataFrames
  • Filtering, sorting, and grouping data
  • Handling missing values and duplicates
  • Merging and reshaping datasets
Module 3 Data Visualisation +
  • Plotting with Matplotlib — line, bar, scatter
  • Statistical plots with Seaborn
  • Customising charts for publication
  • Interactive visualisations with Plotly
  • Saving and exporting charts
Module 4 Statistical Analysis +
  • Descriptive statistics with NumPy and Pandas
  • Hypothesis testing — t-tests, chi-square
  • Correlation and regression analysis
  • Working with SciPy for statistical tests
  • Interpreting and reporting results
Module 5 Automation & APIs +
  • Automating repetitive tasks with scripts
  • Reading and writing files automatically
  • Calling REST APIs and parsing JSON
  • Introduction to web scraping with BeautifulSoup
  • Scheduling and running scripts automatically
Module 6 Introduction to Machine Learning +
  • What is machine learning and when to use it
  • Supervised vs unsupervised learning
  • Building a classification model with Scikit-learn
  • Linear regression for prediction tasks
  • Evaluating and improving model performance

FAQ

Frequently Asked Questions

Do I need programming experience to join? +

No experience needed. The course starts from absolute basics and builds progressively. If you can use a computer you can learn Python.

What software do I need? +

Python is free to install from python.org. We use Jupyter Notebook and VS Code — both free. We guide you through installation in the first session.

Will I get a certificate? +

Yes. All participants who complete the course and practical assessment receive an official ResearchBridge Certificate of Completion.

Can my organisation book this for a team? +

Yes. We offer group pricing for teams of 5 or more. Contact us for a custom quote and we can schedule sessions to suit your team.

How long do I have access to course materials? +

All digital training materials are yours to keep permanently after the course ends.
